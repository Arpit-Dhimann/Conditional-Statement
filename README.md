# Conditional-Statement

This repository is dedicated to mastering **Conditional Logic** and **Control Flow** in Java. Instead of simple calculations, this project focuses on how a program makes decisions based on user input and various conditions.

## 🚀 Projects Included

- **If_Else_Statement.java**: A foundational program to understand basic conditional branching.
- **Else_If_Statement.java**: Implements an "Else-If Ladder" to handle multiple grading criteria based on marks.
- **Switch_Break_Statement.java**: A robust, menu-driven calculator that manages multiple arithmetic operations and includes error handling for "division by zero".

## 🛠️ Concepts Covered
- **Decision Making:** `if`, `else if`, `else` constructs.
- **Branching:** `switch-case` statements with `break` implementation.
- **Input Handling:** Using `java.util.Scanner` for interactive programs.
- **Defensive Programming:** Error handling and edge-case management (e.g., division by zero).
